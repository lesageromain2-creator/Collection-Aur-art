README DU SITE COLLECTION AUR'ART




Ce dépôt contient le code source du site Association Pro – Collection Aur’art, un site web développé en PHP et basé sur une base de données MySQL administrée via phpMyAdmin.
Le projet vise à fournir une plateforme simple, performante et accessible permettant :

- La consultation des articles publiés par l’association ainsi que des annonces événementielles.

- L’authentification sécurisée et la création d’une session unique pour chaque utilisateur.

- La gestion complète du contenu via une interface administrateur (création, édition, suppression d’articles dans des rubriques spécifiques).

- La gestion de l’espace personnel pour chaque membre (modification du profil, ajout d’une description…).

- L’affichage public du profil des membres de l’association.

- La valorisation des œuvres et projets artistiques de la collection Aur’art.

- L’ensemble a été développé dans une architecture simple afin d’être maintenable et évolutif.

  

<> Technologies utilisées

PHP — backend

HTML / CSS / JavaScript — interface utilisateur

MySQL — base de données relationnelle

phpMyAdmin — gestion & administration de la base

Hébergement — serveur compatible PHP/MySQL
